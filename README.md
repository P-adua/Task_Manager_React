# 📝 Task Manager (React)

Gerenciador de tarefas simples desenvolvida com React e Vite.

## 🚀 Funcionalidades

- Adicionar tarefas
- Remover tarefas
- Marcar tarefas como concluídas
- Editar tarefas
- Adicionar tarefa com tecla Enter
- Persistência de dados com localStorage

## 🛠️ Tecnologias utilizadas

- React
- Vite
- JavaScript
- CSS

## 💾 Como rodar o projeto

```bash
npm install
npm run dev
